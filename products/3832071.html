Blockchain application teaching system

Taking the SCT node host as the core, it is equipped with a set of hardware accessories and experimental software required for the course (8 sections) to help the experimental operator combine theory and practice to form a basic blockchain technology knowledge structure.

Based on the core chain technology blockchain intelligent terminal, 3 blockchain intelligent terminals are independently networked, and the intelligent terminal provides main chain services, courseware and tutorial services.

The system software has built-in core chain SCC chain, blockchain browser, SCC soft wallet, SCC chain experiment software, and IoT device interface.
